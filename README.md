CreationBinder
============

サンシャインクリエイションのチェックリストを作成するAndroidアプリ


このアプリについて
------------

**当プロジェクトは「クリエイション事務局」とは一切関係ありません。**

当アプリに関するお問い合わせは「クリエイション事務局」へは送らないようご協力お願いします。

アプリ内の出展サークル情報は[公式サイト](http://www.creation.gr.jp/)で公開されているものを元に作成しております。

開発環境
------------

- AndroidStudio v0.9.+
- JDK 7
